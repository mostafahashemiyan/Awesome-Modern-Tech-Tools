## ü§ñ AI-Powered Tools

* **[Cursor](https://cursor.com)** ‚Äì **The AI-native code editor.** Built as a fork of VS Code, it features deep integration with LLMs, allowing for "Composer" mode where it can write across multiple files simultaneously. 
    * *Why it's awesome:* It understands your entire codebase, not just the file you're currently editing.
* **[Ollama](https://ollama.com)** ‚Äì **Run LLMs locally.** A lightweight, extensible framework to run models like Llama 3, Mistral, and Gemma on your own machine.
    * *Why it's awesome:* Full privacy and zero cost. Works perfectly for building AI apps offline.
* **[v0.dev](https://v0.dev)** ‚Äì **Generative UI by Vercel.** Tell the AI what UI you want, and it generates high-quality React/Tailwind code ready to be shipped.
    * *Why it's awesome:* Turns hours of CSS styling into seconds of prompting.
* **[Aider](https://aider.chat)** ‚Äì **AI Pair Programming in the Terminal.** A command-line tool that lets you edit code in your local git repo. It can refactor, fix bugs, and even write tests.
    * *Why it's awesome:* Works seamlessly with your existing Git workflow.

## ü¶Ä Modern CLI & Rust-based Tools

* **[Bun](https://bun.sh)** ‚Äì **The fast all-in-one JS runtime.** A direct competitor to Node.js that includes a bundler, test runner, and a blazingly fast package manager.
    * *Why it's awesome:* `bun install` is up to 25x faster than `npm install`.
* **[Zed](https://zed.dev)** ‚Äì **High-performance editor.** A GPU-accelerated code editor written in Rust. It‚Äôs built for performance and seamless real-time collaboration.
    * *Why it's awesome:* It opens instantly and feels significantly more responsive than Electron-based editors.
* **[Biome](https://biomejs.dev)** ‚Äì **One toolchain for the web.** It provides formatting and linting for JS, TS, and JSON with 97% compatibility with Prettier but at much higher speeds.
    * *Why it's awesome:* Replaces multiple tools (ESLint, Prettier) with one single, super-fast binary.
* **[Warp](https://www.warp.dev)** ‚Äì **The terminal for the 21st century.** A Rust-based terminal with AI search, collaborative workflows, and a text-editor-like experience.
    * *Why it's awesome:* Command history is grouped into "blocks," making it easy to copy-paste output.

## üåê Web Development & UI

* **[Shadcn/UI](https://ui.shadcn.com)** ‚Äì **Beautifully designed components.** Not a library, but a collection of components that you copy and paste into your project.
    * *Why it's awesome:* You own the code. No more fighting with library-specific CSS overrides.
* **[Astro](https://astro.build)** ‚Äì **The web framework for content.** Known for "Islands Architecture," it allows you to build sites that ship zero JavaScript by default.
    * *Why it's awesome:* Perfect for blogs, portfolios, and documentation with incredible SEO.
* **[Tauri](https://tauri.app)** ‚Äì **Build desktop apps with web tech.** A Rust-based alternative to Electron that results in tiny binaries and minimal memory usage.
    * *Why it's awesome:* Your app can be 3MB instead of 100MB compared to Electron.
* **[Hoppscotch](https://hoppscotch.io)** ‚Äì **Open-source API development.** A fast, lightweight alternative to Postman that runs directly in your browser.
    * *Why it's awesome:* Real-time, transparent, and completely free.
